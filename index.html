Stardew Valley multiplayer host swapper

Want to swap who's a farmhand and who's the host? This handy tool performs the changes described in this guide for you: www.reddit.com/r/StardewValley/comments/8i5z1j/guide_how_to_change_hosts_of_a_multiplayer_save

Always keep back-ups! Modifying save files can break your game or be incompatible with updates. Use at your own risk.



Paste the contents of the file HostCharacterName_123456789 here.

See https://stardewvalleywiki.com/Saves for where to find your save file. Once it's parsed, option buttons will appear. (It can take a minute.)

Results will appear here.


Known issues:

  • Host shown on load screen stays original host when first loading
  • Different people are seeing different amounts of things donated to the community center (I see bundle as complete but you don't)
  • The TV shows different luck levels for different players (it used to always match)
  • If the new host wasn't there when Willy got unlocked, his door stays locked
  • New host may get outdated mail

Issues that should be fixed (7/31/2018):

  • New host repeats host-only events, like picking mushrooms or bats
  • Messes up which house is upgraded
  • Community center and Jojamart events may act as if they didn't happen.
  • Community improvements like minecarts may stop working.
